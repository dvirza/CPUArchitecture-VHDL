קובץ README פרויקט גמר:

TOP (MCU.vhd) – קובץ העוטף את כל המערכת, מקבל את הכניסות ומוציא את היציאות הדרושות, מחבר בין כל הרכיבים הרשומים מטה.

MIPSenv (MIPSenv.vhd) – קובץ הליבה של הCPU בארכיטקטורה הרצויה, יודע לתקשר עם הזיכרונות השונים ולבצע את ההוראות המתקבלות ולהפעיל את שאר המערכת כולל שליחת קווי בקרה.
	MIPScontrol - 	קובץ המתכלל את כל קווי הבקרה שמוציאה הליבה למערכת, יודע לקבל שורת הוראה ולפענח אותה ולאחר מכן להוציא את הסיגנלים הרלוונטים לשאר השלבים.
	MIPSdecode -	קובץ המתכלל את כל נושא הרגיסטרים הפנימי של הליבה, כלומר יודע לקבל הוראה וקווי בקרה ולהעביר את התוכן הרלוונטי לרגיסטר הנכון ובכך להזרים את המידע בתוך המערכת וגם לזיכרון בהמשך דרך קומפוננטות נוספות.
	MIPSdmem - 	קובץ היודע לחבר את הליבה לזיכרון הכללי (לא של ההוראות)
	MIPSexe -		קובץ המכיל את יחידת הביצוע של הליבה כלומר ה ALU, יודע לקבל קווי בקרה ולהמיר אותם לפעולות חישוב הנדרשות ולאחר מכן להעביר את תוכן החישוב לזיכרון או לרגיסטרים בהתאם להוראות ה CONTROL.
	MIPSfetch -	קובץ אשר יודע לתקשר עם זיכרון ההוראות, לקבל אותן "להביא", ולבצע פעולות הסתעפות במידה ונדרש, מחובר לקווי הבקרה הרצויים בCONTROL.
	MIPSintr - 	קובץ אשר יודע לקבל את הפסיקות ממחולל הפסיקות ויודע להתמודד איתן, להעביר לביצוע כשצריך ולכבות או להדליק את קווי הבקרה החוצה.

DIV_env (DIVenv.vhd) – מאיץ חומרה שמבצע חלוקת שלמים לא מסומנים. יודע לתקשר החוצה ולקבל את כל הסיגנלים הרצויים לטובת ביצוע החישוב.
	div - 		קובץ הליבה אשר מבצע את החישובים בפועל, מכיל את הרגיסטרים הנדרשים ועובד עם מונה צעדים, 	מוציא פסיקה שהנתונים מוכנים בסיום.

INTERRUPT_env (INTRenv.vhd) – רכיב היודע לתכלל את סוגי הפסיקות ולהעבירן לCPU ולקבל ממנו קווי בקרה המאפשרים פסיקות אליו. יוצא להוציא בקשת פסיקה ולקבל ACK.
	INTRreg - 	קובץ המכיל את היחידה הקטנה והבסיסית ביותר של קבלת פסיקה, יודע לקבל סיגנל אחד של פסיקה ובעזרת רגיסטר של עליית שעון (כלומר עליית פסיקה מ 0 ל 1) יודע להוציא קו בקרה אחד החוצה המורה על פסיקה.
	INTRseveralregs -	קובץ המחבר 8 יחידות קטנות לכדי רגיסטר אחד בגודל 8 והוא בהמשך יקושר לקווים הרצויים מעלה.
	INTRcore - 	קובץ היודע לקבל את הרגיסטר בגודל 8 ולבצע את תעדופי הפסיקות ובנוסף לכתוב את הTYPE ברגיסטר הרצוי.

GPIO (GPIO.vhd) – רכיב התומך ביציאות וכניסות חיצוניות למיקרו מעבד.
	sw - 			קובץ המאפשר כניסה עבור ה סוויצים בעזרת bdirpin.
	decode - 		קובץ המבצע decode עבור 7 segment עבור ה HEX
	hex_led -		קובץ המאפשר קבלת והוצאות מידע עבור ה HEX ו ה LEDS בעזרת bdirpin .
	IOaddrDecode-קובץ המבצע CS, בחירה של כל אחד מהGPIO לפי כתובת ב addr.

BTimer_env (BTimerEnv.vhd) – רכיב הממש טיימר פנימי של המיקרו מעבד, הרכיב יודע להוציא אות ריבוע מסוג PWM. מתכלל בתוכו תקשורת מהadd_bus ויוצר קשר עם הרכיבים הפנימיים יותר.
	pwm -		קובץ המתכלל את כל נושא הוצאת האות המרובע, מקבל תוכן של רגיסטרים ועל ידיהם מחשב את האות המרובע לפי הלוגיקה הרצויה בפרויקט.
	BTimer - 		קובץ המכיל את מחלקי השעונים, המוקס לבחירת הפסיקה ואת הרגיסטרים של השעון ורגיסטרי הקבלה של המידע.
